# PHP Job Queue

A lightweight job queue system built with pure PHP and MySQL.

This project allows you to dispatch, store, and process background jobs using a command-line worker.

## Features

- Simple and lightweight architecture
- MySQL-based job storage
- Command-line worker
- Job dispatching system
- Job status management
- Retry attempts support
- Exception handling
- Environment configuration using `.env`
- Eloquent Query Builder support

---

## Requirements

- PHP 8.2 or higher
- MySQL 8.0 or higher
- Composer

---

## Installation

> Composer installer (Recommended)

```bash
composer create-project arefshojaei/job-queue-system my-app
```

> Clone the repository:

```bash
git clone https://github.com/ArefShojaei/PHP-job-queue.git
```
